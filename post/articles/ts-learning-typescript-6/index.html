[TIL] 러닝타입스크립트 - 6. 배열

Posted by Hyewon Kang on Fri, Apr 28, 2023

⚠️ 러닝 타입스크립트 책의 모든 내용을 담지 않으며 주관적인 기준으로 필요한 내용만 작성하고, 부족한 내용에 대해 추가 개념을 정리합니다.

튜플

  • 고정된 크기의 배열

  • 예) let yearAndWarrior: [number, string];

  • 가변 길이의 배열 타입은 튜플 타입에 할당할 수 없다. → TS는 생성된 배열을 튜플이 아닌 가변 길이의 배열로 취급

  • 나머지 매개변수로서의 튜플 (p.126)

    1function logPair(name: string, value: number) {
2    console.log(`${name} has ${value}`);
3}
4
5const pairArray = ['Amage', 1]; // 이 경우 타입은 (string | number)[] 가 된다.
6logPair(...pairArray); // Error: 확산 인수는 튜플 유형을 가지거나 나머지 매개 변수로 전달되어야 합니다.
7
8const pariTupleCorrect: [string, number] = ['Amage', 1];
9logPair(...pariTupleCorrect);

튜플 추론

  1. 명시적 튜플 타입
1function firstCharAndSizeExplicit(input: string): [string, number] {
2    return [input[0], input.length];
3}
  1. const 어서션
  • as const 연산자 이용
  • const 어서션은 타입스크립트에 타입을 유추할 때 읽기 전용(read-only)이 가능한 값 형식을 사용하도록 지시한다.
1// 타입 : readonly [1157, "Tomoe"]
2const readonlyTuple = [1157, 'Tomoe'] as const;
  • const 어서션은 유연한 크기의 배열을 고정된 크기의 튜플로 전환하는 것을 넘어서, 해당 튜플이 읽기 전용이고 값 수정이 예상되는 곳에서 사용할 수 없음을 나타낸다.
